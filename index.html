Recent Projects

MyLincoln App

Client Inrix
Role Lead front-end UI developer
Tech HTML, CSS, SASS, Backbone.js, Phonegap, JavaScript

Description

Specified, implemented, and enforced UI development guidelines, to ensure consistency within the codebase. Worked with the design team to translate design visuals in to UI templates, for use with Backbone.js.

Challenges

Ensuring that an HTML5 application feels like a native app on both Android and iOS. Maintaining responsiveness whilst retrieving live traffic data from external APIs, over mobile networks of varying quality.

Solution

Component based templates for backbone.js allow re-use and simplified code making it easier to troubleshoot. Added conditional logic in the build for some device specific overrides to prevent too many collisions in properties. Worked closely with services team.
