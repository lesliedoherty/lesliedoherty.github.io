Animations

iPad Animated Background**

Revisiting this: CSS Keyframe is a crab.

Based on the animation property in the media object. I decided to make use of transform: rotateZ(360deg) to use the GPU power more than the CPU since it was running high at ~40% in Chrome.

The conditional is used inside the for project loop. v-if="project.media.animation"

Next we evaluate the kind of animation frame we are using: v-if="project.media.animation.frame === 'iPad'

Finally we pass the correct props to the component: :image="project.media.animation.image"
